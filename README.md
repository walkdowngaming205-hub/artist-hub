# 🎵 Artist Hub

Artist Hub is a React-based music platform that allows artists to showcase, organize, and manage their music in a clean and responsive web application.

## Features

* Browse artist songs
* View song details
* Responsive mobile-friendly design
* React Router navigation
* Component-based architecture
* Future audio player integration

## Technologies Used

* React
* JavaScript (ES6+)
* HTML5
* CSS3
* React Router
* Git & GitHub

## What I Learned

This project helped me strengthen my skills in:

* Building reusable React components
* Managing application state
* Routing and navigation
* Front-end application structure
* Git version control

## Installation

```bash
npm install
npm run dev
```
